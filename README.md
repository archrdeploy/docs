# Archr documentation

Mintlify site for [archr.win](https://archr.win). Archr is a self-serve token launchpad on Robinhood Chain (chainId 4663) with burned Uniswap V4 liquidity, creator rewards paid in ETH, an optional dev-buy, and a time-locked airdrop program for ARCHR holders.

This directory is the source of truth for the Archr documentation site. Build and publishing are handled by the maintainers' internal release process.

## Local preview

```bash
npm i -g mint
mint dev
```

Run from this directory (`docs/mintlify`), where `docs.json` lives.
